
Объекты в fxml-документе могут быть созданы следующими путями:
    1. С помощью публичного безаргументного конструктора класса - см. пакет object.bean:
        <VBox> </VBox>
       Главное условие: класс должен соответствовать JavaBean - конструкции (иметь
       безаргументный публичный конструктор и правильно именнованные свойства с соответствующими
       геттерами и сеттерами)

    2. С помощью статического метода valueOf() - см. пакет object.value

    3. С помощью фабричного метода - см. пакет object.factory

    4. С помощью билдера - см. пакет object.builder. Используется в случае, если класс
        не соответствует JavaBean - конструкции.

    5. Если класс реализует Map интерфейс, то объект может быть создан напрямую,
      а егосвойства будут инициализироваться спомощью get() и put() методов класса - см. пакет object.map
